Objects

You can have a group of data that are related in some way and instead of using a variable for which one, an object can host all of them

Objects can be described as collections of related properties, where which property is represented as a key value pair

To indicate a object curly brace {} is used on a variable, also known as the object literal syntax

Example in Variable in Object
rice pack variable name property key
1 Kg variable value property value

The dot notation can be used on the property's of the variable

Or it can be comma delimited

Example dot notation:
var storeManager = {}
storeManager.age = 45;
storeManager.weight = 90;

Example comma delimited:
var assistantManager = {
age: 45,
weight: 90,
experience: 1
}

Example brackets notation:
var assistantManager = {}
assistantManager["age"] = 45;
assistantManager["weight"] = 90;
assistantManager["experience"] = 1;

The brackets allow to

  • Use space on the propety keys
    • apartment["number of rooms"] = 3;
  • Use numbers (as the string data type) as property keys [not recommended]
    • car["2021"] = 20000;
  • Evaluate expressions

Example: apartment["number of rooms"] = 3;

Math object cheat sheet

Number constants

  • Math.PI: PI number which is approximately 3.14159
  • Math.E: Euler's number which is approximately 2.718
  • Math.LN2: natural logarithm of 2 which is approximately 0.693
  • Math.random: generate a random number between 0 and 0.99

Rouding methods (arredondar)

  • Math.ceil(): rounds up to the closest integer and only upwards
  • Math.floor(): rounds down to the closest integer
  • Math.round(): rounds up to the closest integer if the decimal is .5 or above; otherwise, rounds down to the closest integer
  • Math.trunc(): trims the decimal, leaving only the integer
  • Math.pow(2,3): calculates the number 2 to the power of 3, the result is 8 (pow=potência -> 2 elevado ao cubo(3))
  • Math.sqrt(16): calculates the square root of 16, the result is 4 (raiz quadrada)
  • Math.cbrt(8): finds the cube root of 8, the result is 2 (raiz cúbica)
  • Math.abs(-10): returns the absolute value, the result is 10 (resultado absoluto que cálcula a distência do valor até zero independente de ser positivo ou negativo)
  • Logarithmic methods
    • Math.log(): returns the natural logarithm (base e) of x
    • Math.log2(): returns the base-2 logarithm of x
    • Math.log10(): returns the base-10 logarithm of x
  • Math.min(9,8,7), Math.max(9,8,7): return the minimum and maximum values of all the inputs = 7 and 9 in the example
  • Math.sin(), Math.cos(), Math.tan(), etc.: trigonometric methods

Strings

.concat = concat method adds any value to the string it was running

charAt(index): return the character that corresponds to the index position asked

indexOf(""): returns the location of the first position that matches a character

lastIndexOf(""): returns the location of the last position that matches a character

split(""): chops up the string based on the character selected, which will disappear on the result, into an array of sub-strings with the rest

toUpperCase(): change the casing of all letter of the stringto upper case

toLowerCase(): change the casing of all letter of the stringto lower case
