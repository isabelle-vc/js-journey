Functions

A functions is used for repeat a set of actions with or without parameters

Without parameters = function example()

With parameters = function example(a, b, c, d)

There isn't a limit for how many parameters a fuction can have

Arrays

To buid one, the Array Literal Syntax can be used. This means a open square bracket [] to identify the collection of items as a group

Example: var example1 ["example1", "example2", "example3"]

The items placement inside the array, are numbered automatically starting at 0, this is also called the value's index

Value Index
example 1 0
example 2 1
example 3 2

This values can be accessed with their index number. It can be log when using console.log(`index: ${i}`)

When a parameter is used, JS doesn't require that the data type be specified, it will use whatever was given if valid to the function

Objects

You can have a group of data that are related in some way and instead of using a variable for which one, an object can host all of them

Objects can be described as collections of related properties, where which property is represented as a key value pair

To indicate a object curly brace {} is used on a variable, also known as the object literal syntax

Example in Variable in Object
rice pack variable name property key
1 Kg variable value property value

The dot notation an be used on the property's of the variable

Or it can be comma delimited

Example dot notation:
var storeManager = {}
storeManager.age = 45; storeManager.weight = 90;

Example comma delimited:
var assistantManager = { age: 45, weight: 90, experience: 1 }
storeManager.age = 45; storeManager.weight = 90;

Example brackets notation:
var assistantManager = { age: 45, weight: 90, experience: 1 }
storeManager.age = 45; storeManager.weight = 90;

The brackets allow to

  • Use space on the propety keys
    • apartment["number of rooms"] = 3;
  • Use numbers (as the string data type) as property keys [not recommended]
    • car["2021"] = 20000;
  • Evaluate expressions

Example: apartment["number of rooms"] = 3;

New properties can be added to the previous ones. To do that dot notation or brackets notation can be used

The same way it can be used to update a already existing property

storeManager.nextAchievement = "open a new store"

assistantManager.nextAchievement = "got promoted"
