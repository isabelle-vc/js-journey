Booleans

Checks if a statment is true or false, which are the only two possible values as "maybe" doesn't exist in this case

&& = AND
console.log(true && true) will output: true
console.log(true && false) will output: false
console.log(false && true) will output: false
console.log(false && false) will output: false

|| = OR
console.log(true || true) will output: true
console.log(true || false) will output: true
console.log(false || true) will output: true
console.log(false || false) will output: false

! = NOT
console.log(true ! true) will output: true
console.log(true ! false) will output: true
console.log(false ! true) will output: true
console.log(false ! false) will output: false

Concatenation operator uses the same symbol as Addition operator (+), but the concatenation is when it's used with strings

Example: [console.log (365 + "days")]
In this case the JS convertes the number value into a string value so that it can run the (+) operator
This process is called "coercion"
Sometimes it can show an unexpected result
Example: [console.log (1 + "2")]
If this happens, the number value will be transformed into a string value and the result will be "12" and not the addition operation

+= as Addition assignment operator
It's used when one wants to accumulate the values stored in a variable

Operator precedence and associativity
Precedence considers the math logic of operations where, the multiplication and division are calculated first than addition and Subtraction
Associativity determines how the precedence works when the code uses operators with the same precedence
Example: [5 > 4 > 3]
The [5 > 4] is evaluated first (to `true`), then true > 3 is evaluated to `false`, because the `true` value is coerced to `1`

Looping

Used for repeatedly execute a block of code until a certain condition is satisfied

Incrementer or loop counter, represented by the letter "i" in programming, without it the loop doesn't when to start or end, which is a infinite loop

Types of loops

For: loop a block of code a certain number of times

Set the value: [var i = 0]

Specify the condition: [i < 3] (the end-count value)

Increment the counter: [i++] (increase or decrease)

While: loop a block of code a certain number of times too

Set the value: [var i = 0] before the loop

Specify the condition: [(i < 3)] is set outside of the loop (the end-count value), and if the information is true, it will run the loop

Increment the counter: [i=i+1] (increase or decrease) is set inside the loop's body

The [i++] means the same as [i=i+1]

Nested: process more than one data at the same time

In this cases exists the outer loops and the inner loops

Be careful with the use of loops, as multiple levels of nested loops will slow the code
